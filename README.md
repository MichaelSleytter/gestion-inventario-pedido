## Resumen

El presente proyecto consiste en el desarrollo de un sistema web de
gestión de inventario y pedidos para una distribuidora comercial,
utilizando Java Spring Boot, MySQL y arquitectura REST.

## Abstract

This project consists of the development of a web-based inventory
and order management system for a commercial distributor, using
Java Spring Boot, MySQL and REST architecture.

---

## Realidad Problemática

La distribuidora comercial objeto de estudio presenta problemas
críticos en la gestión manual de su inventario, incluyendo pérdidas
por descontrol de stock, registros físicos poco confiables y ausencia
de trazabilidad en los movimientos de productos.

## Justificación

El desarrollo de este sistema permitirá automatizar y digitalizar
los procesos de inventario y pedidos, reduciendo errores humanos,
mejorando la trazabilidad y optimizando la toma de decisiones.

---

## Objetivo General

Desarrollar un sistema web de gestión de inventario y pedidos para
una distribuidora comercial que permita optimizar el control de stock
y la trazabilidad de movimientos mediante una plataforma digital.

## Objetivos Específicos

1. Implementar un módulo de gestión de productos con control de stock.
2. Desarrollar un módulo de pedidos con seguimiento de estados.
3. Registrar entradas y salidas de inventario con trazabilidad completa.
4. Gestionar usuarios con roles diferenciados.
5. Documentar e integrar una API REST con Swagger/OpenAPI.
